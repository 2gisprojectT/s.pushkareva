Тест-кейсы: 
	№1
	Название: 
		Поиск рейсов при некорректном выборе даты
	
	Предусловия: 
		1. Открыта главная страница сайта https://www.onetwotrip.com
		2. Корректно(см. документацию) заполнено поле «Откуда»
		3. Корректно(см. документацию) заполнено поле «Куда»
		4. Отображается календарь для выбора даты в поле «Туда»
	
	Шаги:
		1. Установить фокус на поле «Туда»
		2. Выбрать вчерашнее число 
		3. Нажать кнопку «Найти»
	
	Ожидаемый результат:
		Кнопка «Найти» заблокирована


	№2
	Название: 
		Отображение списка городов после ввода первых 3х допустимых символов
	
	Предусловия
		1. Существует минимум 1 город на вводимую допустимую последовательность
		2. В городе есть аэропорт
		3. Открыта страница сайта https://www.onetwotrip.com
		4. Поле «Откуда» не заблокировано
	
  	Шаги: 
		1. Установить фокус на поле «Откуда» 
		2  Ввести в выбранное поле допустимую последовательность(см. документацию) из трех букы
  
  	Ожидаемый результат:  
		 Отображение списка городов, название которых начинается на введенную допустимую посоедовательность


	№3
  	Название: 
    		Проверка доступа  к заказам по электронному номеру билета и фамилии пассажира
  
  	Предусловия: 
		1. Существует оформленный и оплаченный заказ. (Присутствует номер электронного билета)
		2. Зарегистрирован пассажир с фамилией Ivanov
		3. Открыта страница сайта https://www.onetwotrip.com
		4. Выбрано поле «Мои заказы»
  
	 Шаги:
		1. Установить фокус на поле «Номер электронного билета»
		2. Ввести зарегистрированный номер билета для пассажира «Ivanov»(см. докуентацию)
		3. В поле «Фамилия пассажира» фамилию «Ivanov»
		4. Нажать кнопку «Найти»
  
  	Ожидаемый результат: 
		Билет найден. Данные по заказу найдены и отображаются. Присутствует полное имя пассажира, номер электронного билета, паспортные данные, дата и время покупки билета, дата и время вылета, город отправления, город назначения, категория авиационного билета(эконом,премиум-эконом,бизнес), поле для возможности обмена и возврата билета.
