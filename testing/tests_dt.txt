Тест-кейс №1
	Название:
		Получение кода подтверждения для заказа такси
	Предусловия:
		1. Открыта главная страница сайта http://novosibirsk.rutaxi.ru
		2. Корректно(см в документации) заполнены поля адреса отправления(поля  «Откуда», «№ дома», «Подъезд»)
		3. Корректно(см. в документации) заполнены поля адреса назначения(поля  «Куда», «№ дома», «Подъезд»).
		4. Нажата кнопка «Заказать»
	Шаги:
		1. Ввести корректный номер телефона(см. в документации) в поле «Введите ваш номер телефона»
		2. Нажать на кнопку «Получить код подтверждения»
	Ожидаемый результат: 
		На введенный номер поступит 6-значный числовой код подтверждения. Откроется форма, содержащая поле вводе кода подтвержден			ия заказа.
